# AI Recipe Generator - Contexto del Proyecto

## ¿Qué estamos construyendo?
Una aplicación web que genera recetas personalizadas usando IA basándose en los ingredientes que el usuario tiene disponibles.

## Objetivo del proyecto
- **Portfolio:** Demostrar habilidades Full Stack + AI Integration
- **Mercado laboral:** Especializarse en desarrollo con IA para trabajo remoto
- **Tecnologías:** React + Node.js + OpenAI API (JavaScript puro, sin TypeScript)

## Funcionalidades principales
1. **Input de ingredientes** - Usuario ingresa ingredientes disponibles
2. **Opciones de personalización** - Tipo de cocina, tiempo de preparación, dificultad
3. **Generación con IA** - OpenAI API genera receta completa
4. **Visualización de receta** - Ingredientes, pasos, tiempo estimado
5. **Guardar favoritas** - localStorage para persistencia

## Stack técnico elegido
- **Frontend:** React + CSS (sin frameworks CSS por ahora)
- **Backend:** Node.js + Express
- **IA:** OpenAI API (GPT-4)
- **Base de datos:** localStorage inicial, después MongoDB
- **Deploy:** Netlify (frontend) + Railway/Render (backend)

## Contexto del desarrollador
- **Nivel:** Full Stack junior
- **Experiencia:** HTML, CSS, JavaScript, React
- **Aprendiendo:** WordPress, PHP (básico)
- **NO sabe:** TypeScript (por eso usamos JavaScript puro)
- **Objetivo:** Conseguir trabajo remoto en desarrollo con IA

## Tiempo estimado
- **Total:** 1 semana (7 días)
- **Dedicación:** 8 horas por día
- **Enfoque:** Proyecto funcional para portfolio
